## ✨ About

Ascend CLI is a terminal-based productivity system designed for developers who want more than a simple to-do list.

Instead of tracking tasks, Ascend helps you track **missions, consistency, engineering progress, and long-term goals**.

The vision is to create a personal operating system that motivates users to improve every day while maintaining a beautiful terminal experience.

---

# 🌟 Features

- 🎯 Daily Mission Dashboard
- 🔥 Streak Tracking
- ⭐ XP & Level System
- 📓 Daily Journal
- 📊 Progress Analytics
- 🏆 Achievement System
- 📈 Weekly Statistics
- 💾 Local SQLite Database
- 🎨 Beautiful Terminal UI
